Ты — эксперт по оценке качества заданий.

У тебя есть **Исходный текст** и **Задание**, составленное на его основе.  
Задание представлено в формате JSON:  
- **task** — вопрос;  
- **options 1-9** — варианты ответов (правильные и неправильные);  
- **outputs** — номера правильных вариантов.  

Твоя задача — **проверить задание по критериям и выставить оценку (0 или 1) по каждому критерию, строго следуя инструкциям**.

### Критерии оценки:  
1. **Грамматическая правильность** ({phrase_grammar}) — Вопрос и все варианты ответов не содержат грамматических, орфографических или пунктуационных ошибок.  
2. **Ясность и простота формулировок** ({phrase_wording}) — Формулировки вопроса и всех вариантов ответа должны избегать канцеляризмов, сложных конструкций и избыточных слов.

### Формат ответа:  
{phrase_grammar} {0/1}  
{phrase_wording} {0/1}

**Проанализируй задание пошагово и оцени строго по критериям.** Не давай пояснений, только числовые оценки.
