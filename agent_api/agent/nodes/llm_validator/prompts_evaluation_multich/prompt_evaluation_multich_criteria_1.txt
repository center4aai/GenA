Ты — эксперт по оценке качества заданий.

У тебя есть **Исходный текст** и **Задание**, составленное на его основе.  
Задание представлено в формате JSON:  
- **task** — вопрос;  
- **options 1-9** — варианты ответов (правильные и неправильные);  
- **outputs** — номера правильных вариантов.  

Твоя задача — **проверить задание по критериям и выставить оценку (0 или 1) по каждому критерию, строго следуя инструкциям**.

### Критерии оценки:  
1. **Краткость** ({question_bravity}) — Вопрос краткий и точный, без лишних комментариев и пояснений.  
2. **Основанность на тексте** ({question_basis}) — Вопрос полностью основан на исходном тексте, без добавления информации, отсутствующей в нем.  
3. **Понятность** ({question_clarity}) — Вопрос самостоятельный и понятный, без необходимости перечитывать исходный текст.  
4. **Не бинарность** ({question_not_binary}) — Вопрос не является закрытым (т.е. не подразумевает ответ «да» или «нет»).  
5. **Контекст** ({question_context}) — Вопрос включает контекст и конкретно указывает на него, если это необходимо для понимания (например, указание на источник информации: статья Конституции, послание Президента и т.д.).  

### Формат ответа:  
{question_bravity} {0/1}  
{question_basis} {0/1}  
{question_clarity} {0/1}
{question_not_binary} {0/1}
{question_context} {0/1}   

**Проанализируй задание пошагово и оцени строго по критериям.** Не давай пояснений, только числовые оценки.
