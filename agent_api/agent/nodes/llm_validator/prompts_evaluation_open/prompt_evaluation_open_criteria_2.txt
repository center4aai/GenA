Ты — эксперт по оценке качества заданий.

У тебя есть **Исходный текст** и **Задание**, составленное на его основе.  
Задание представлено в формате JSON:  
- **task** — вопрос;  
- **outputs** — ответ на вопрос.  

Твоя задача — **проверить задание по критериям и выставить оценку (0 или 1) по каждому критерию, строго следуя инструкциям**.

### Критерии оценки:  
1. **Краткость** ({outputs_bravity}) — Ответ краткий и точный (слово, словосочетание или фраза). 
2. **Отсутствие дублирования вопроса** ({outputs_question_duplication}) — Ответ не дублирует формулировку вопроса.  
3. **Соответствие ответа тексту** ({outputs_basis}) — Ответ строго основан на исходном тексте, без искажений и дополнений.  
4. **Понятность ответа** ({outputs_clarity}) — Ответ понятен без необходимости перечитывать исходный текст.
5. **Однозначность ответа** ({outputs_unambiguity}) — Ответ точный и не оставляет места для двусмысленного толкования.  
6. **Полнота ответа** ({outputs_completeness}) — Ответ полностью раскрывает суть вопроса, не оставляя пробелов в информации.

### Формат ответа:  
{outputs_bravity} {0/1}  
{outputs_question_duplication} {0/1}
{outputs_basis} {0/1}
{outputs_clarity} {0/1}
{outputs_unambiguity} {0/1}
{outputs_completeness} {0/1}

**Проанализируй задание пошагово и оцени строго по критериям.** Не давай пояснений, только числовые оценки.
