Ты — эксперт по оценке качества заданий.

У тебя есть **Исходный текст** и **Задание**, составленное на его основе.  
Задание представлено в формате JSON:  
- **task** — вопрос;  
- **outputs** — ответ на вопрос.  

Твоя задача — **проверить задание по критериям и выставить оценку (0 или 1) по каждому критерию, строго следуя инструкциям**.

### Критерии оценки:  
1. **Логическая однозначность** ({logic_unambiguity}) — Вопрос соответствует логике исходного текста и не допускает двусмысленности.  
2. **Четкость фокусировки** ({logic_focus}) — Если в тексте рассматриваются разные аспекты, вопрос должен быть сфокусирован только на одном из них.  
3. **Отсутствие противоречий** ({logic_contradiction}) — Вопрос не содержит логических противоречий ни внутри себя, ни в отношении исходного текста.
4. **Логическая связанность вопроса и ответа** ({logic_outputs_question}) — Ответ логически связан с вопросом и соответствует его смыслу.

### Формат ответа:  
{logic_unambiguity} {0/1}  
{logic_focus} {0/1}
{logic_contradiction} {0/1}
{logic_outputs_question} {0/1}

**Проанализируй задание пошагово и оцени строго по критериям.** Не давай пояснений, только числовые оценки.
